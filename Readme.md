# Agentic AI Proof-of-Concept

This project showcases an autonomous multi-agent system that reviews, modifies, tests, and deploys code using AI agents.

## Agents
1. Review Agent
2. Modification Agent
3. Test Script Generator
4. Test Executor
5. Deployment Agent
